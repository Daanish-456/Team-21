# Stone & Soul Website Repository

An ethical jewelery website created using Laravel and MySQL.

By Team 21

## Features

- Stock Management: adding/removing/modifying products, changing stock amounts
- Admin Dashboard: can view orders, contact tickets, stock and general statistics
- Can order products
- Wishlists
